# Housing Area Insight Agent｜租屋資訊 Agent

這是一個租屋資訊專案。
使用者可以輸入想了解的租屋地點，系統會整理該區域附近的生活資訊，協助使用者在租屋前做初步評估。

目前系統以網頁方式呈現，使用者可以在頁面中輸入地點，查看該地區的基本資訊，也可以透過聊天機器人詢問更細節的問題。

---

## 專案功能

目前系統可以提供以下資訊：

* 地點解析：將使用者輸入的地點轉換為可查詢的位置資訊
* 租金範圍：查詢指定區域附近的租金統計資料
* 交通資訊：整理附近交通站點與交通便利性
* 生活機能：查詢附近超商、醫療、公園、停車場等生活設施
* 夜間生活：分析附近深夜營業店家、24 小時店家與宵夜便利性
* 空氣品質：查詢鄰近空品測站資料，作為長期居住環境參考
* 聊天機器人：使用者可以直接詢問特定地點或生活需求

---

## 使用方式

使用者可以在網頁輸入地點，例如：

```text
OO車站
OO大學
```

系統會整理該區域的生活圈資訊。

也可以在聊天機器人中輸入更具體的問題，例如：

```text
OO大學附近半夜有東西吃嗎？
```

```text
中壢區租金和桃園區租金哪個比較低？
```

```text
中央大學和中原大學誰的生活機能比較好？
```

## 環境設定

先建立 `.env` 檔案，依照 `.env.example` 填入需要的 API key。

```env
GOOGLE_API_KEY=
GOOGLE_PLACES_API_KEY=
GOOGLE_MAPS_API_KEY=
TDX_CLIENT_ID=
TDX_CLIENT_SECRET=

ADK_BASE_URL=http://127.0.0.1:8000
ADK_APP_NAME=housing_agent
ADK_USER_ID=web_user
```

請勿將 `.env` 上傳至 GitHub。

## 安裝方式

建立虛擬環境：

```bash
python -m venv .venv
```

啟動虛擬環境：

```bash
.venv\Scripts\Activate.ps1
```

安裝套件：

```bash
pip install -r requirements.txt
```

---

## 啟動方式

本專案需要開啟兩個 terminal。

### 1. 啟動 ADK Agent

```bash
adk web --port 8000
```

### 2. 啟動網頁後端

```bash
python app.py
```

啟動後開啟瀏覽器：

```text
http://127.0.0.1:5001
```

## 後續規劃

未來可延伸「通勤範圍分析」功能，讓使用者不只以單一地點作為租屋搜尋基準，也能從日常通勤需求出發，評估更符合生活動線的租屋區域。

例如：

```text
我要找台大附近騎車 15 分鐘以內的房子
```

系統可根據使用者指定的目的地、通勤方式與可接受時間，推估適合的租屋搜尋範圍，並進一步整理該範圍內的租金、交通與生活機能資訊。

透過此功能，租屋分析可以從「某個地點附近」延伸為「在可接受通勤時間內的生活圈」，讓使用者能以更貼近日常移動需求的方式評估租屋選擇。
